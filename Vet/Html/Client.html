Funcionalidades

A principal função do VET é a medição da pulsação do animal para saber qual o seu estado. Para melhorar a precisão, os dados serão guardados numa base de dados para assim ser possível observar o histórico do animal e no final concluir a análise.

Dependências

Para que o dispositivo funcione corretamente é necessário que existem as seguintes condições:

  • 1. Conexão á internet;
  • 2. WebBrowser funcional.

Ambiente de Operação

A aplicação deve permitir o utilizador aceder a toda a informação armazenada na base de dados. Tanto a aplicação como o dispositivo móvel devem comunicar com a base de dados e são acessíveis em qualquer lugar.
